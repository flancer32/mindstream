# Runtime Architecture — AGENTS

Path: `./ctx/docs/architecture/runtime/AGENTS.md`

## Назначение

Документ фиксирует границы и состав подуровня `runtime` архитектурной документации, описывающего модель запуска и разделение ответственности между bootstrap-слоем и приложением без перехода к реализации или эксплуатационным процедурам.

## Карта уровня

- `AGENTS.md` — текущий документ уровня, задающий границы и состав подуровня runtime.
- `bootstrap.md` — декларативное описание архитектурной модели запуска backend-приложения и границы ответственности между bootstrap-слоем и приложением.

## Границы уровня

Подуровень `runtime` описывает архитектурные инварианты запуска, передачу управления и допустимые границы ответственности между средой исполнения, bootstrap-слоем и приложением, не вводя требований к деплою, CI/CD, инфраструктуре, пользовательским сценариям и технологическим деталям реализации.

## Связи с другими уровнями

Подуровень `runtime` конкретизирует архитектурную форму, определённую в `ctx/docs/architecture/AGENTS.md`, и использует продуктовый смысл из `ctx/docs/product/` как словарь понятий, оставаясь в границах архитектурного слоя и не подменяя ограничения из `ctx/docs/constraints/`.
