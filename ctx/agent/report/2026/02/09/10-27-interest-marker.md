# Отчёт итерации — interest-marker

## Резюме изменений
- Добавлен UI-маркер совпадения интересов с вертикальной полосой и процентом в карточках публикаций.
- Вынесена интеграция со scoring attention в отдельный модуль-адаптер.
- Обновлены стили ленты для размещения маркера и анимации заполнения.

## Детали работ
- Создан модуль адаптера для инициализации attention, расчёта и кэшированных значений score без прямого импорта attention из UI: `web/ui/js/interest-score.mjs`.
- Перестроен рендер карточек ленты, добавлены элементы маркера, карта ссылок на DOM-узлы и обновление значений при пересчёте: `web/ui/js/feed.mjs`.
- Добавлены стили для сетки контента, вертикального маркера и подписи процента: `web/ui/css/feed.css`.

## Результаты
- Каждая карточка отображает вертикальный маркер совпадения и процент под ним.
- Заполнение маркера отражает `score ∈ [0..1]` с плавным переходом и обновляется после пересчёта интересов.
- UI-код не импортирует attention напрямую и использует адаптер для получения score.
