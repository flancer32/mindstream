# Отчёт итерации

## Резюме изменений

Storage-схема и SchemaManager обновлены для векторных эмбеддингов pgvector с фиксированной размерностью, добавлена генерация расширения `vector` и соответствующие unit-тесты.

## Детали работ

- Обновлена декларативная схема эмбеддингов на `vector(1536)` и увеличена версия схемы: `src/Storage/Schema.mjs`.
- SchemaManager поддерживает тип `vector`, создаёт `CREATE EXTENSION IF NOT EXISTS vector` перед созданием таблиц и корректно обрабатывает значения при восстановлении: `src/Storage/SchemaManager.mjs`.
- Добавлен unit-тест для проверки `vector(1536)` и расширения: `test/unit/back/Storage/SchemaManager.test.mjs`.

## Результаты

- Эмбеддинги хранятся как векторные данные pgvector фиксированной размерности.
- Схема создаёт расширение `vector` перед таблицами.
- Unit-тесты выполнены: `npm run test:unit`.
