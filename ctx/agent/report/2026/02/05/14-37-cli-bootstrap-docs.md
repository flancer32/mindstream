# Отчёт итерации — cli-bootstrap-docs

## Резюме изменений
- Упрощён bootstrap в `bin/app.mjs` до модели «всегда останавливать приложение после возврата из команды».
- Обновлена нормативная документация CLI в `ctx/docs/code/cli` в соответствии с новой моделью завершения.

## Детали работ
- В `bin/app.mjs` оставлена единая схема `run → stop → process.exit(code)` без специальной ветки для `runtime:*`, при этом runtime-команды по норме не возвращают управление.
- В `ctx/docs/code/cli/dispatcher.md` уточнена модель освобождения ресурсов и поведения runtime-команд.
- В `ctx/docs/code/cli/overview.md` зафиксировано, что bootstrap завершает приложение после возврата управления.

## Результаты
- Bootstrap и документация согласованы с требованием «команда завершает приложение при возврате управления».
- Нормативная модель CLI остаётся валидной для runtime-команд: возврат управления трактуется как нештатная ситуация.
