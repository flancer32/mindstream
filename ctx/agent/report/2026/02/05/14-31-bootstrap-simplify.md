# Отчёт итерации — bootstrap-simplify

## Резюме изменений
- Упрощён bootstrap в `bin/app.mjs` до формы с `try/catch/finally` при сохранении нормативной модели CLI и разделения runtime/engineering сценариев.
- Сохранено гарантированное освобождение ресурсов для инженерных CLI-команд и корректная реакция на нештатный возврат управления из runtime-команды.

## Детали работ
- Переписан lifecycle: `run → stop → process.exit(code)` для инженерных команд в `finally`.
- Для runtime-команд `stop()` вызывается только при возврате управления, с предупреждением и без управления exit code как сигналом жизненного цикла.
- Ошибки `run()` и `stop()` логируются на уровне bootstrap через `Mindstream_Shared_Logger`, без изменения диспетчера.

## Результаты
- Bootstrap соответствует `ctx/docs/code/cli/dispatcher.md` и `ctx/docs/code/cli/overview.md`.
- CLI-команды корректно освобождают ресурсы и завершают процесс в инженерном режиме.
