# Отчёт итерации — cli-tests

## Резюме изменений
- Добавлены unit-тесты для новых CLI-классов: корневого диспетчера, db-диспетчера и команды `db:schema:create`.
- Тесты проверяют маршрутизацию, обработку ошибок, вызов зависимостей и отказ от аргументов.

## Детали работ
- Создан `test/unit/back/Cli.test.mjs` с изолированными сценариями через DI-контейнер в test mode.
- Добавлены стаб-объекты для логгера, диспетчеров и `SchemaManager` без статического импорта production-кода.

## Результаты
- Новый тестовый файл: `test/unit/back/Cli.test.mjs`.
- Тесты: `npm run test:unit` (успешно).
