# Отчёт итерации: anonymous identity + attention beacons

## Резюме изменений

В frontend добавлена анонимная идентичность (UUID) с явной активацией через меню, отправка регистрации `/identity` и дублирование новых сигналов внимания в `/attention` через `navigator.sendBeacon`. Меню показывает UUID после активации и хранит его в `localStorage`.

## Детали работ

- Добавлен модуль `web/ui/js/identity.mjs` для генерации UUID v4, хранения в `localStorage`, отправки `/identity` и `/attention` beacons, а также синхронизации между вкладками.
- `web/ui/js/attention.mjs` дополняет фиксацию внимания отправкой сигнала на backend при наличии identity.
- `web/ui/js/feed.mjs` получил меню активации идентичности (кнопка `*`) с русским подтверждением и отображением полного UUID; регистрация identity отправляется при наличии UUID.
- `web/ui/css/feed.css` расширен стилями для меню идентичности.
- Возможное расхождение: явная активация идентичности в UI может пересекаться с инвариантом отсутствия управления режимами доступа из `ctx/docs/composition/ui.md`.

## Результаты

- Код обновлён: `web/ui/js/identity.mjs`, `web/ui/js/attention.mjs`, `web/ui/js/feed.mjs`, `web/ui/css/feed.css`.
- Тесты не запускались (изменения только в frontend, обязательные команды для `src/` не применимы).
