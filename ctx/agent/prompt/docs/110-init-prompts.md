# Дополнительные промпты для создания начального корпуса документов

## Отдельный документ

{{ctx/docs/output.md}}

Мы работаем с `PATH_TO_DOC_MD`. Я приложил текущую документацию по продукту. Подумай, какие вопросы ты хочешь мне задать, чтобы корректно создать этот документ для MVP. Задай эти вопросы.

## AGENTS.md

{{ctx/docs/{area}/output.md}}

Прочитай текущий корпус документов. Сделай `AGENTS.md` для текущей области корпуса документов.

## Итоговая проверка

{{ctx/docs/{area}/output.md}}

Проверь итоговый корпус документов в текущей области на декларативность, непротиворечивость, полнота, связность, плотность, компактность, неизбыточность, отсутствие описаний очевидного. В конце дай кроткое резюме.
